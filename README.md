For the music feature to work properly, you'll need to add your own romantic piano melody audio file named `romantic-music.mp3` in the same directory as `index.html`.

Alternatively, you can use a royalty-free music service or generate a simple romantic melody.

## Customization

### Change Valentine's Day Date

Edit line 66 in `script.js`:

```javascript
const valentinesDay = new Date("2026-02-14T00:00:00");
```

### Add More Poems

Add to the `poems` array starting at line 227 in `script.js`.

### Change Color Scheme

Modify CSS variables in `styles.css` (lines 4-12):

```css
:root {
  --deep-red: #8b1538;
  --rose-red: #c41e3a;
  --blush-pink: #ffb6c1;
  /* ... more colors */
}
```
